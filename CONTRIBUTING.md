# Contributing

This is a **private fork** of [Postiz](https://github.com/gitroomhq/postiz-app) maintained by [Eteya](https://eteya.ai).

## We don't accept external PRs here

If you want to contribute to Postiz itself, please submit upstream: https://github.com/gitroomhq/postiz-app

## Internal contributors (Eteya team)

- Branches: feature-branches off `main`, merged via PR.
- Commits: Conventional Commits (`feat(scope):`, `fix(scope):`, etc).
- Before commit: `pnpm lint && pnpm typecheck`.
- Update [FORK-CHANGELOG.md](./FORK-CHANGELOG.md) when changing custom code.
- See [AGENTS.md](./AGENTS.md) for full conventions.

## Reporting bugs

- **Eteya-specific issues:** open a GitHub issue here.
- **Postiz core bugs:** report upstream at [gitroomhq/postiz-app/issues](https://github.com/gitroomhq/postiz-app/issues).
- **Security issues:** see [SECURITY.md](./SECURITY.md). Do NOT open public issues for security.

## Code of conduct

Be kind. Be specific. No spam.
